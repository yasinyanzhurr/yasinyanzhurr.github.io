HTML Dasar

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. Mari belajar dasar-dasar HTML untuk memulai perjalanan Anda dalam web development!

Daftar Isi
  • 1. Pengenalan HTML
  • 2. Struktur Dasar HTML
  • 3. Elemen-elemen HTML
  • 4. Latihan Membuat Halaman

1. Pengenalan HTML

HTML adalah fondasi dari setiap halaman web. Ini adalah bahasa markup yang menggunakan tag untuk mendefinisikan elemen-elemen dalam halaman web.

Apa yang bisa dibuat dengan HTML?
  • Struktur halaman web
  • Teks dan paragraf
  • Gambar dan multimedia
  • Link dan navigasi
  • Form dan input

2. Struktur Dasar HTML

Setiap dokumen HTML memiliki struktur dasar berikut:


<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Halo, Dunia!</h1>
    <p>Ini adalah paragraf.</p>
</body>
</html>

3. Elemen-elemen HTML

Tag Fungsi Contoh
<h1> - <h6> Heading (Judul) <h1>Judul Utama</h1>
<p> Paragraf <p>Ini paragraf</p>
<a> Link <a href="url">Link</a>

4. Latihan Membuat Halaman

Tugas: Buat halaman profil sederhana

Gunakan kode berikut sebagai template:


<!DOCTYPE html>
<html lang="id">
<head>
    <title>Profil Saya</title>
</head>
<body>
    <h1>[Nama Anda]</h1>
    <p>[Deskripsi singkat tentang Anda]</p>
    <h2>Hobi:</h2>
    <ul>
        <li>[Hobi 1]</li>
        <li>[Hobi 2]</li>
    </ul>
</body>
</html>
