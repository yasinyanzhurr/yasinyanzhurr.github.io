CSS Dasar

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan halaman web. Dengan CSS, kita bisa membuat website menjadi lebih menarik dan responsif.

Daftar Isi
  • 1. Pengenalan CSS
  • 2. Selector CSS
  • 3. Properties CSS Dasar
  • 4. Box Model
  • 5. Latihan Styling

1. Pengenalan CSS

CSS dapat ditambahkan ke HTML dengan 3 cara:

1. Inline CSS

<p style="color: blue; font-size: 16px;">Teks biru</p>
2. Internal CSS

<style>
    p {
        color: blue;
        font-size: 16px;
    }
</style>
3. External CSS

<link rel="stylesheet" href="styles.css">

2. Selector CSS

Selector Contoh Deskripsi
Element p { } Memilih semua elemen <p>
Class .merah { } Memilih elemen dengan class="merah"
ID #header { } Memilih elemen dengan id="header"

3. Properties CSS Dasar

Warna dan Background

.box {
    color: blue;
    background-color: yellow;
}
Font dan Teks

.text {
    font-size: 16px;
    font-family: Arial;
    text-align: center;
}

4. Box Model

Box Model adalah konsep dasar dalam CSS yang menjelaskan bagaimana elemen HTML ditampilkan sebagai box dengan properti berikut: 


.box {
    margin: 10px;      /* Jarak luar */
    border: 1px solid black;  /* Garis tepi */
    padding: 15px;     /* Jarak dalam */
    width: 300px;      /* Lebar konten */
    height: 200px;     /* Tinggi konten */
}

5. Latihan Styling

Tugas: Styling Card Profil

Gunakan CSS untuk membuat card profil menarik:


<div class="profile-card">
    <img src="avatar.jpg" alt="Profile">
    <h3>John Doe</h3>
    <p>Web Developer</p>
</div>
                                            

.profile-card {
    /* Tambahkan CSS di sini */
}
                                            

.profile-card {
    width: 300px;
    padding: 20px;
    text-align: center;
    border-radius: 10px;
    box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}

.profile-card img {
    width: 100px;
    height: 100px;
    border-radius: 50%;
    margin-bottom: 15px;
}

.profile-card h3 {
    color: #333;
    margin-bottom: 5px;
}

.profile-card p {
    color: #666;
}
