Admin Dashboard User Guide

Getting Started

  1. Log in to the admin dashboard using your credentials
  2. Navigate through different sections using the sidebar menu
  3. View statistics and recent activities on the main dashboard

Managing Posts

  • Create new posts using the "New Post" button
  • Edit existing posts from the posts list
  • Schedule posts for future publication
  • Manage post categories and tags

Handling Comments

  • View and moderate comments from the Comments section
  • Approve or reject pending comments
  • Reply to user comments
  • Enable/disable comments for specific posts

Analytics & Reports

  • View page views and user engagement metrics
  • Track popular posts and topics
  • Monitor subscriber growth
  • Export analytics reports
