Database MySQL

MySQL adalah sistem manajemen database relasional yang populer. Pelajari cara membuat dan mengelola database dengan MySQL untuk aplikasi web Anda!

1. Pengenalan MySQL

Database Relasional

MySQL menyimpan data dalam tabel yang saling berhubungan, memungkinkan pengelolaan data yang efisien.

SQL Queries

Menggunakan bahasa SQL untuk manipulasi dan pengambilan data.

2. Membuat Database


-- Membuat database baru
CREATE DATABASE blog_db;

-- Menggunakan database
USE blog_db;

-- Membuat tabel users
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) NOT NULL UNIQUE,
    email VARCHAR(100) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
Struktur Tabel users:
Field Type Null Key
id INT NO PK
username VARCHAR(50) NO UQ
email VARCHAR(100) NO

3. Tabel dan Relasi


-- Membuat tabel posts
CREATE TABLE posts (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT,
    title VARCHAR(200) NOT NULL,
    content TEXT,
    FOREIGN KEY (user_id) REFERENCES users(id)
);

-- Menambahkan indeks
CREATE INDEX idx_user_id ON posts(user_id);

4. Operasi CRUD

Create (INSERT)

INSERT INTO users (username, email) 
VALUES ('john_doe', 'john@example.com');
Read (SELECT)

SELECT username, email 
FROM users 
WHERE id = 1;
Update (UPDATE)

UPDATE users 
SET email = 'new.email@example.com' 
WHERE username = 'john_doe';
Delete (DELETE)

DELETE FROM users 
WHERE id = 1;

5. Latihan Query

Tugas: Menampilkan Posts dengan Info User

Buat query untuk menampilkan judul post dan username pembuatnya:


-- Tulis query di sini
SELECT p.title, u.username
FROM posts p
_____ JOIN users u
ON p.user_id = u.id;
                                        

SELECT p.title, u.username
FROM posts p
INNER JOIN users u
ON p.user_id = u.id;

Error yang Sering Terjadi

Error: Access denied for user 'username'@'localhost'

Solusi: Periksa username dan password MySQL Anda, pastikan user memiliki hak akses yang sesuai.

Error: Table 'database.table' doesn't exist

Solusi: Pastikan nama tabel sudah benar dan database yang digunakan aktif.
