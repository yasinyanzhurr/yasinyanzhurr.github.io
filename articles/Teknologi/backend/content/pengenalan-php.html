Pengenalan PHP

PHP adalah bahasa pemrograman server-side yang populer untuk pengembangan web. Mari pelajari dasar-dasar PHP untuk memulai perjalanan Anda dalam backend development!

1. Apa itu PHP?

Server-Side

PHP dijalankan di server, bukan di browser pengguna. Ini memungkinkan pemrosesan data yang aman dan kompleks.

Database Integration

PHP memiliki dukungan yang kuat untuk berbagai database, terutama MySQL.

2. Syntax Dasar

File PHP harus diakhiri dengan ekstensi ".php"


<?php
    // Ini adalah komentar
    echo "Hello, World!";  // Menampilkan teks
    
    /* Ini adalah komentar
       multi-baris */
    
    // PHP tags
    ?>
Output:

Hello, World!

3. Variabel dan Tipe Data


<?php
    // String
    $nama = "John";
    
    // Integer
    $umur = 25;
    
    // Float
    $tinggi = 175.5;
    
    // Boolean
    $isActive = true;
    
    // Array
    $hobi = ["membaca", "coding", "gaming"];
    
    // Menampilkan variabel
    echo "Nama: " . $nama;
    echo "<br>";
    echo "Umur: " . $umur;
?>
Output:

Nama: John
Umur: 25

4. Latihan Coding

Tugas: Membuat Kalkulator Sederhana

Buat program PHP untuk menambahkan dua angka:


<?php
    // Fungsi penjumlahan
    function tambah($a, $b) {
        return $a + $b;
    }
    
    // Test fungsi
    $angka1 = 10;
    $angka2 = 5;
    $hasil = tambah($angka1, $angka2);
    
    echo "$angka1 + $angka2 = $hasil";
?>
Output:

10 + 5 = 15
